Cursos de Programación

HTML & CSS

Aprende a construir y diseñar páginas web desde cero. HTML estructura el contenido y CSS le da estilo visual.

JavaScript

Agrega interactividad a tus sitios web. Aprende lógica, eventos, animaciones y manipulación del DOM.

Python

Lenguaje versátil para automatización, desarrollo web, ciencia de datos e inteligencia artificial.

React

Biblioteca de JavaScript para crear interfaces modernas, rápidas y dinámicas con componentes reutilizables.

Node.js

Permite usar JavaScript en el servidor. Ideal para crear APIs, aplicaciones en tiempo real y backend escalable.

SQL

Lenguaje para gestionar bases de datos. Aprende a almacenar, consultar y organizar información de forma eficiente.

Git & GitHub

Control de versiones y colaboración en proyectos. Aprende a guardar cambios, trabajar en equipo y publicar tu código.

Inteligencia Artificial

Introducción al aprendizaje automático, redes neuronales y cómo las máquinas pueden tomar decisiones inteligentes.

Flutter

Framework de Google para crear aplicaciones móviles nativas para Android y iOS con una sola base de código.

Seguridad Informática

Aprende a proteger sistemas, redes y datos. Conoce los fundamentos de ciberseguridad y buenas prácticas.
